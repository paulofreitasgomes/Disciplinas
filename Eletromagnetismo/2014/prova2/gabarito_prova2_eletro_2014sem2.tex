\noindent
{\color{blue}1a) Solução:} Dados: esfera de raio $R$ e potencial na sua superfície $V_0 (\theta) = k \cos 3 \theta$. Pede-se o potencial em todo o espaço $V (r,\theta)$. O potencial é obtido através da eq. de Laplace, cuja solução geral (considerando separação de variáveis) é:
 \begin{equation}
 V (r,\theta) = \left\{
\begin{array}{rl}
 V_d (r,\theta) = \sum\limits_{n=0}^{\infty} A_n r^n P_n (\cos \theta), & \text{se  } r \leq R  \\
\\
V_f (r,\theta) =\sum\limits_{n=0}^{\infty} \dfrac{B_n}{r^{n+1}} P_n (\cos \theta),  & \text{se  } r \geq R
\end{array} 
\right.
 \label{potencialgeral}
\end{equation} 
Os termos $P_n(\cos \theta)$ são os conhecidos polinômios de Legendre. A forma dentro e fora da esfera é resultado das condições de contorno, na qual o potencial tem que ir a zero tanto para $r$ pequeno quanto para $r$ no infinito. Resta então encontrar os coeficientes $A_n$ e $B_n$. Para isso, precisamos impor as condições de continuidade no potencial ao longo da superfície da esfera em $r=R$. Isso por que o potencial é contínuo ao longo de uma superfície mesmo quando há densidade de carga (Lei de Gauss). Assim, impondo continuidade no intervalo $0 \leq r\leq R$ na eq. \ref{potencialgeral}, implica que $V_d$ deve ser contínuo desde $r=0$ até $r=R$, onde o potencial é dado. Logo:
\begin{equation}
V_d (R,\theta) = \sum\limits_{n=0}^{\infty} A_n R^n P_n (\cos \theta) = V_0 (\theta) \label{continuidadermenorR}
\end{equation}
Multiplicando os dois lados desta equação por $P_m(\cos \theta) \sin \theta$ e integrando em $d\theta$, teremos:
\begin{equation}
\sum\limits_{n=0}^{\infty}  \int_0^{\pi} A_n R^n P_n (\cos \theta) P_m(\cos \theta) \sin \theta d\theta = \int_0^{\pi} V_0 (\theta)  P_m(\cos \theta) \sin \theta d\theta \nonumber
\end{equation}
A integral é resolvida considerando que os polinômios de Legendre formam um conjunto ortogonal:
\begin{equation}
 \int_0^{\pi} P_n  (\cos \theta ) P_m  (\cos \theta ) \sin \theta d\theta  = \left\{
\begin{array}{rl}
 0,& \text{se  } n \neq m  \\
\\
 \dfrac{2}{2n+1},& \text{se  } n = m
\end{array} 
\right.
 \label{ortogonalidade}
\end{equation}
Assim, a integral é zero para todos os termos no qual $m \neq n$, ficando apenas o termo $n=m$. Isolando $A_m$ da eq. resultante temos:
\begin{equation}
A_m = A_n = \dfrac{2n+1}{2R^n} \int_0^{\pi} V_0(\theta) P_n (\cos \theta) \sin \theta d\theta
\end{equation} 
Esse é o procedimento normal de obtenção dos coeficientes em uma série. 

Para encontrar $B_n$, podemos usar o mesmo procedimento: impor a continuidade no intervalo $r \geq R$
\begin{equation}
V_f (R,\theta) =\sum\limits_{n=0}^{\infty} \dfrac{B_n}{R^{n+1}} P_n (\cos \theta) = V_0 (\theta) \label{dkfjeiowowpqq}
\end{equation}
resultando em uma integral. Porém, há um meio mais simples, também impondo a continuidade. Das eqs. \ref{continuidadermenorR} e \ref{dkfjeiowowpqq}, temos:
 \begin{equation}
 V_d (R,\theta) = V_f (R,\theta) \nonumber
 \end{equation}
Abrindo as expressões de $V_d$ e $V_f$, obtemos uma igualdade de séries de $P_n$. 
\begin{equation}
\sum\limits_{n=0}^{\infty} A_n R^n P_n (\cos \theta) = \sum\limits_{n=0}^{\infty} \dfrac{B_n}{R^{n+1}} P_n (\cos \theta) \nonumber
\end{equation}
Assim, a igualdade apenas será verdade se os respectivos termos de cada $P_n$ forem iguais:
\begin{equation}
A_n R^n= \dfrac{B_n}{R^{n+1}}  \quad \quad \Longrightarrow  \quad \quad  B_n = A_nR^{2n+1} \label{iieieieieie}
\end{equation}
Uma vez escrito os coeficientes em função de $V_0$, vamos encontrá-los. Para facilitar o cálculo de $A_n$, vamos escrever $V_0$ em função de $P_n$:
\begin{equation}
V_0 (\theta) = k \cos 3 \theta = \dfrac{k}{5} \left[ 8P_3 (\cos \theta ) - 3 P_1 (\cos \theta ) \right] 
\nonumber
\end{equation}
Analisando a ortogonalidade de $P_n$ (eq. \ref{ortogonalidade}), vemos que apenas os coeficientes $A_1$ e $A_3$ vão ser diferentes de zero. De fato, calculando:
\begin{eqnarray}
A_0 &=& \dfrac{k}{5} \int_0^{\pi} \left[ 8P_3 (\cos \theta ) - 3 P_1 (\cos \theta ) \right] P_0 (\cos \theta) \sin \theta d\theta = 0 \nonumber \\
A_1 &=& \dfrac{3}{2R} \dfrac{k}{5} \int_0^{\pi} \left[ 8P_3 (\cos \theta ) - 3 P_1 (\cos \theta ) \right] P_1 (\cos \theta) \sin \theta d\theta = -\dfrac{9k}{10} \int_0^{\pi} P_1 P_1 \sin \theta d\theta = - \dfrac{3k}{5R} \nonumber \\
A_2 &=& \dfrac{5}{2R^2} \dfrac{k}{5} \int_0^{\pi} \left[ 8P_3 (\cos \theta ) - 3 P_1 (\cos \theta ) \right] P_2 (\cos \theta) \sin \theta d\theta = 0 \nonumber \\
A_3 &=& \dfrac{7}{2R^3} \dfrac{k}{5} \int_0^{\pi} \left[ 8P_3 (\cos \theta ) - 3 P_1 (\cos \theta ) \right] P_3 (\cos \theta) \sin \theta d\theta = 8\dfrac{7k}{10R^3} \int_0^{\pi} P_3 P_3 \sin \theta d\theta = \dfrac{8k}{5R^3} \nonumber \\
A_4 &=& A_5 = A_6 = ... = 0 \nonumber
 \end{eqnarray}
Agora podemos encontrar $B_n$ a partir da eq. \ref{iieieieieie}:
\begin{equation}
B_1 = A_1 R^3 = - \dfrac{3}{5}kR^2 \quad \quad \therefore \quad \quad B_3 = A_3 R^7 = \dfrac{8}{5}kR^4 \quad \quad \therefore \quad \quad  A_0 = A_2 = A_4 = A_5 =... = 0
\nonumber 
\end{equation}
Assim, a solução fica então:
\begin{equation}
V (r,\theta) = \left\{
\begin{array}{rl}
 A_1 r P_1 (\cos \theta ) + A_3 r^3 P_3 (\cos \theta )= {\color{blue}\boxed{-\dfrac{3k}{5R} r P_1 (\cos \theta ) + \dfrac{8k}{5R^3} r^3 P_3 (\cos \theta )}}, & \text{se  } r \leq R  \\
\\
 \dfrac{B_1}{r^2} P_1 (\cos \theta ) + \dfrac{B_3}{r^4} P_3 (\cos \theta ) = {\color{blue}\boxed{-\dfrac{3kR^2}{5r^2}  P_1 (\cos \theta ) + \dfrac{8kR^4}{5r^4} P_3 (\cos \theta )}},  & \text{se  } r \geq R
\end{array} 
\right. \nonumber
\end{equation}
Para o potencial dado, apenas 2 termos na expansão foram necessários.

{\color{blue}1b) Solução:} Para calcular a densidade de carga, devemos usar mais uma condição de continuidade do potencial. Ainda da Lei de Gauss, a derivada na direção normal a superfície do potencial é descontínua em uma superfície com carga:
\begin{equation}
 \left(  \dfrac{\partial V_d }{\partial r}- \dfrac{\partial V_f }{\partial r} \right)\biggr\rvert_{r=R}   = -\dfrac{\sigma(\theta)}{\varepsilon_0}  \nonumber
\end{equation}
Repare que a derivada normal do potencial é exatamente negativo do campo elétrico. No caso, $r$ é a coordenada normal a superfície da esfera. Substituindo a solução geral (eq. \ref{potencialgeral}) nesta eq. e já usando os coeficientes encontrado, teremos:
\begin{eqnarray}
\sigma (\theta) &=& \varepsilon_0 \sum_{n=0}^{\infty} (2n+1) A_n R^{n-1} P_n (\cos \theta ) = \varepsilon_0 \left[  (2+1) A_1 R^0 P_1(\cos \theta ) + 7 A_n R^2 P_3(\cos \theta ) \right] \nonumber \\
&=& {\color{blue}\boxed{\dfrac{\varepsilon_0 k}{5R} \left[ 56 P_3 (\cos \theta )-9 P_1 (\cos \theta ) \right] }}
\end{eqnarray}

{\color{blue}2) Solução}: A primeira vista parece um problema simples de movimento uniformemente variado. O detalhe que impede isso é que a força não é constante, pois depende da posição. Dessa forma não podemos usar as equações do MUV. Precisamos então encontrar a solução do problema $z(t)$, que é a posição em função do tempo. Para isso devemos usar a Lei de Newton. Consideremos a carga na posição $(0,0,d)$ no eixo $z$, e o plano em questão sendo o plano $xy$. Dessa forma a força que a carga sente (dada na prova) é:
\begin{equation}
\vec{F} = -\dfrac{1}{4\pi \varepsilon_0} \dfrac{q^2}{(2z)^2}\hat{z}  \nonumber
\end{equation}
Queremos o tempo necessário para a partícula se deslocar de uma posição inicial $(0,0,d)=d\hat{z}$ até uma posição final $(0,0,0)$. A única força atuante na partícula é a força elétrica. Escrevendo a Lei de Newton em módulo teremos:
\begin{equation}
m \dfrac{d^2z}{dt^2} = -\dfrac{1}{4\pi \varepsilon_0} \dfrac{q^2}{(2z)^2}  
\quad \quad \therefore \quad \quad
\dfrac{d^2z}{dt^2} =- \dfrac{A}{z^2} \label{elllalla}
\end{equation}
onde $A = \dfrac{q^2}{16\pi \varepsilon_0 m}$. Queremos encontrar $z(t)$ a partir desta equação diferencial. Porém a eq. \ref{elllalla} não está escrita em uma forma na qual podemos integrar duas vezes e encontrar $z(t)$, pois a força também depende de $z$. Precisamos então de artifícios matemáticos para reescrever esta equação em uma forma mais direta, na qual possamos integrar. Temos então que mudar a variável de integração. Uma tática é multiplicar a eq. \ref{elllalla} pela velocidade da partícula $v = \dfrac{dz}{dt}$:
\begin{equation}
\dfrac{dz}{dt} \dfrac{dv}{dt} = -\dfrac{A}{z^2}\dfrac{dz}{dt} \nonumber
\end{equation} 
O lado esquerdo e o lado direito desta equação são respectivamente:
\begin{equation}
\dfrac{dz}{dt} \dfrac{dv}{dt} = v \dfrac{dv}{dt} = \dfrac{d}{dt} \left( \dfrac{v^2}{2} \right)  \quad \quad \therefore  \quad \quad -\dfrac{A}{z^2}\dfrac{dz}{dt} = \dfrac{d}{dt} \left( \dfrac{A}{z} \right) \nonumber
\end{equation} 
Logo, ambos os termos são na verdade a derivada temporal de algo:
\begin{equation}
\dfrac{d}{dt} \left( \dfrac{v^2}{2} \right) =  \dfrac{d}{dt} \left( \dfrac{A}{z} \right)  \nonumber
\end{equation}
Se uma função monotônica avaliada em dois pontos retorna valores iguais, é por que os dois pontos são na verdade iguais:
\begin{equation}
f(a) = f(b) \quad \quad \Longrightarrow  \quad \quad a = b \nonumber
\end{equation}
O mesmo vale para a derivada, que também é uma função: quando a derivada de uma coisa é igual a derivada de outra coisa, as duas coisas são iguais, somada uma constante $C$. Logo:
\begin{equation}
\dfrac{v^2}{2} = \dfrac{A}{z} + C \label{woeiufjs} \nonumber
\end{equation}
já que a derivada de qualquer constante é zero. Além disso, essa constante é importante para satisfazer a condição inicial. No instante inicial $t=0$, temos $v=0$ e $z=d$. Usando isto na eq. anterior teremos $C = -\dfrac{A}{d}$. Usando esse valor de $C$ na eq. \ref{woeiufjs} teremos:
\begin{equation}
v^2 = 2A \left( \dfrac{1}{z} -\dfrac{1}{d} \right) = \dfrac{2A}{d} \dfrac{d-z}{z}  \nonumber
\end{equation}
Agora isolamos $v =dz/dt$ para enfim podermos escrever a equação anterior
\begin{equation}
v = \dfrac{dz}{dt} = -\sqrt{\dfrac{2A}{d} \dfrac{(d-z)}{z}} \nonumber
\end{equation}
O sinal negativo é permitido, já que $(\pm \sqrt{b})^2=b$ para qualquer $b$. Porém, aqui a solução negativa é escolhida pois $z$ diminui com o tempo, saindo de $d$ até zero. Agora sim a eq. anterior é direta o suficiente permitindo sua integral. Rearranjando os termos e montando a integral:
\begin{equation}
\int_d^0 \sqrt{\dfrac{z}{d-z}}dz = -\sqrt{\dfrac{2A}{d}} \int_0^t dt   \nonumber
\end{equation}
A integral em $t$ é direta enquanto que integral em $z$ é resolvida usando a fórmula dada na prova. Logo $-d\dfrac{\pi}{2} =-\sqrt{\dfrac{2A}{d}}t$. Rearranjando:
\begin{equation}
t = d\dfrac{\pi}{2} \sqrt{\dfrac{d}{2A}} = d\dfrac{\pi}{2} \sqrt{\dfrac{d}{2} \dfrac{16\pi \varepsilon_0 m}{q^2} } =\dfrac{d\pi}{q}\sqrt{2\pi d \varepsilon_0 m} = 
{\color{blue}\boxed{\sqrt{\dfrac{2\pi^3 d^3 \varepsilon_0 m}{q^2}}}} \nonumber
\end{equation}
